¿Qué es Bigdata?


Para comenzar con la aproximación de este concepto es importante nombrar que el término big data toma fuerza en la década de los 2000 cuando el analista de la industria, Doug Laney, articuló la definición actual de grandes datos como las tres V (volumen, velocidad y variedad), este concepto hace referencia a los datos que generalmente son muy grandes y complejos de procesar con los métodos tradicionales. (sas, Bigdata. Qué es y por qué es importante, https://www.sas.com/es_co/insights/big-data/what-is-big-data.html).

BigData y las industrias 4.0


Para comenzar este tema es importante reconocer el contexto e importancia de la industria 4.0 o cuarta revolución industrial y cómo esto afecta a las empresas en la actualidad. Consiste en la interconexión de todas las partes de una empresa o proceso productivo teniendo como resultado procesos de automatización efectiva, la interconectividad y el manejo de los datos en tiempo real y por ende, empresas mucho más inteligentes.

Esta revolución está marcada por la masificación de tecnologías como la robótica, la analítica de datos, la inteligencia artificial, las tecnologías cognitivas, la nanotecnología y el Internet of Things (IoT), entre otros, lo cual ha forzado a las organizaciones a comprender las dinámicas de sus sectores y servicios, y articularlas con estas tecnologías con el fin de mantener sus posiciones en el mercado.

Dentro de las características de la imagen anterior mostraremos algunas de estas; en el caso del volumen se hace referencia a una cantidad incalculable de datos o información que es generada cada segundo, esto comprende información emitida desde redes sociales, tarjetas de crédito, teléfonos inteligentes etc; la velocidad ya entra como el movimiento que se le da a estos datos, y permite evidenciar qué tan rápido se van produciendo o cambiando estos datos; otra de las características es la variedad, está enmarcada dentro de las diferentes formas o tipos de datos que se van almacenando, estos pueden llegar a ser estructurados o no estructurados.

La inteligencia de negocios

La minería de datos

Fundamentos:

La minería de datos o también llamado Data Mining consiste en la detección de información en grandes cantidades de datos, buscando encontrar relaciones que de otra manera serían insospechadas. Al aplicar las técnicas de minería de datos, se podrán descubrir patrones que sean capaces de resumir y hacer más comprensible grandes cantidades de información, haciéndolas relevantes para cualquier proceso o decisión en la empresa.


Para continuar con la información de la minería de datos, haga clic sobre cada botón “+” en la imagen:

La minería de datos es diferente al big data, para entenderlo podemos decir que el big data es la montaña de información y el Data Mining es la herramienta que se utiliza para excavar dicha montaña con el propósito de encontrar los pedacitos de oro, es decir los hallazgos en conocimiento de gran valor para la empresa.

De manera específica, el proceso típico de minería de datos se desarrolla a través de los siguientes pasos, haga clic en cada número para conocerlos:



Base de Datos: Se selecciona el conjunto de datos, en lo referente a las variables objetivo (con las que se quiere inferir, predecir o calcular), y a las variables independientes (utilizadas para el cálculo o proceso), también lo relacionado con el muestreo de los registros disponibles.

Limpieza e integración: Se analizan las propiedades de los datos y se depura la información identificando la presencia de valores atípicos y ausencia de datos (valores nulos).

Selección y Transformación: la transformación del conjunto de datos de entrada se realizará de diversas formas de acuerdo con el análisis realizado previamente, preparándolo para la aplicación de la técnica de minería de datos considerada como la más apropiada para los datos y para el problema. Este paso también es conocido como preprocesamiento de los datos.

Minería de Datos: se seleccionará y aplicará la técnica de minería de datos, para proceder construir el modelo predictivo, de clasificación o segmentación.

Patrones: se extrae el conocimiento mediante una técnica de minería de datos, en la cual se puede obtener un modelo de conocimiento, que representará los patrones de comportamiento observados de acuerdo con las variables del problema o las relaciones entre las variables.

Evaluación y Presentación: una vez extraído el modelo, se procede a realizar su validación, con el fin de comprobar la validez de las conclusiones obtenidas. En el caso de haber utilizado diferentes técnicas, esto significa que se han obtenido varios modelos (uno por cada técnica), por lo tanto, se deben comparar los modelos definiendo cuál se ajusta mejor al problema. Si no se alcanzan los resultados esperados, se deberán realizar modificaciones en alguno de los pasos anteriores para obtener nuevos modelos.
